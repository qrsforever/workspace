名词:

    横断面研究(Cross-Sectional Study)
        https://en.wikipedia.org/wiki/Cross-sectional_study
        在某一特定时间对某一定范围内的人群，以个人为单位收集和描述人群的特征以及疾病或健康状况。
    
    纵贯研究(Longitudinal Study)
        http://wikipedia.org/wiki/Longitudinal_study
        在一段时间内反复观察同一群人。


统计本身不能说明什么，它的功能只是辅助判断，一个数据集能作统计，能估计参数，但不一定能证明什么


其他:
    Hist: 直方图, 频数
    Pmf; 概率质量函数, 概率(0 - 1)


期望, 均值: 一对双胞胎, 被不同人家收养, 期望用来描述概率分布
率分布: 在统计图中表示概率，横轴是数据的值，纵轴是横轴上对应数据值的概率

二项分布: 发生次数N固定, 关心成功n次的概率
几何分布: 尝试多次, 第一次成功的概率
泊松分布: 某时间范围内, 发生某件事情的概率, 不需要进行一系列试验


指数分布: 独立事件发生的事件间隔(X) ---> 每个时间点上发生的概率相同, 间隔时间的分布
