解压到当前路径下`clips_core_source_630, clips_core_source_640`

编译630
=======

build lib:
    1. `cd clips_core_source_630/core`
    2. `ln -s ../makefiles/makefile.lib makefile`
    3. `make`

build clips:
    1. `cd clips_core_source_630/core`
    2. `ln -s ../makefiles/makefile.gcc makefile`
    3. `make`

编译640
=======

1. `cd clips_core_source_640/core`
2. `make`
